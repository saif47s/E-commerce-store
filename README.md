## Getting Started

Project ko apne local system par setup aur run karne ke liye in commands ko follow karein:

```bash
# 1. Repository ko clone karein
git clone [https://github.com/saif47s/saif-store.git](https://github.com/saif47s/saif-store.git)

# 2. Project directory mein jayen
cd saif-store

# 3. Dependencies install karein
npm install
# ya agar aap yarn use karte hain: yarn install

# 4. Local development server start karein
npm run dev
# ya yarn dev
